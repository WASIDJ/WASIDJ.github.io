1. 随机事件及其概率

1.1 利用“四大公式“求事件概率

[!note] 加法公式 设 $P(A) = 0.5$，$P(A \cup B) = 0.8$，且 $A$ 和 $B$ 互不相容，求 $P(B)$。 $P(A \cup B) = P(A) + P(B) - P(AB)$ $P(A \cup B \cup C) = P(A) + P(B) + P(B) - P(AB) - P(AC) -P(BC) + P(ABC)$ PS:加奇减偶 ${A B C} \subset {A B}\Rightarrow P(ABC) \leq P(AB)$

[!NOTE] 减法公式⭐ $P(A - B) = P(A) - P(AB)$ 对立事件: $P(\overline{A}) = 1 - P(A)$

[!NOTE] 条件概率⭐ 条件概率公式:$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$ $P(A \overline{B}) = P(A)-P(AB)$ 证:$P(A \overline{B}) = P{A(\Omega - B)}=P(A-AB)$
